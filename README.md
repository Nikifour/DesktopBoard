# Desktop Board

Минимальный MVP приложения-доски для рабочего стола Windows.

## Что уже есть

- полноэкранное окно без рамки;
- компактная верхняя панель с иконками, группами и tooltip;
- контекстное меню по правой кнопке мыши для создания и редактирования элементов;
- карточки заметок, задач и расписания;
- карточки изображений, видео и URL/сайтов;
- URL/сайт растягивается вместе с размером карточки;
- иконка типа элемента в правой части каждого хедера;
- соединения между элементами и свободными точками: линия, стрелка, обратная стрелка, две стрелки;
- цвет соединений задается глобально в настройках и отдельно через контекстное меню линии;
- соединения можно выделять, добавлять путевые точки двойным щелчком и перетаскивать эти точки;
- выделение нескольких элементов рамкой левой кнопкой мыши;
- масштабирование доски колесом мыши;
- перемещение камеры средним кликом;
- притягивание элементов к фоновой сетке при перемещении;
- путевые точки и свободные точки соединений тоже притягиваются к сетке;
- перетаскивание карточек;
- изменение размера карточек;
- групповые подложки: элементы внутри подложки двигаются вместе с ней;
- окно настроек с языком интерфейса, цветом фона, цветом соединений, цветами новых элементов, snap-to-grid и hotkeys;
- секция цветов новых элементов в настройках раскрывается и по умолчанию свернута;
- цвет верхней панели и окна настроек подстраивается под системный accent color;
- переключатель темы интерфейса: системная, светлая, темная;
- индивидуальный цвет хедера и фона для каждого элемента через контекстное меню;
- локальное сохранение состояния в `userData` Electron;
- режим просмотра/редактирования;
- горячая клавиша `Ctrl+Alt+B` для показа/скрытия окна;
- горячая клавиша `Ctrl+Alt+L` для закрепления/редактирования доски.

## Управление

- Колесо мыши: приблизить/отдалить доску.
- Средняя кнопка мыши: двигать рабочую поверхность.
- `Соединение`: включить режим создания линии. Кликните старт, затем финиш; финиш может быть элементом или пустой точкой доски.
- `Ctrl` + левая кнопка по элементам: выбрать несколько нодов по одному.
- Правая кнопка мыши по выбранным нодам: `Соединить` создаст линию между ними. Для трех и более выбранных нодов создается цепочка.
- Левая кнопка по соединению: выделить линию.
- Двойной щелчок по выделенному соединению: добавить путевую точку.
- Перетаскивание путевой точки: изменить маршрут линии.
- Правая кнопка мыши по соединению: изменить вид стрелок, цвет или удалить линию.
- Левая кнопка мыши по пустому месту + протягивание: выделить элементы рамкой.
- Поле групповой подложки тоже можно использовать как стартовую область для рамочного выделения.
- Левая кнопка мыши за хедер/drag-зону: двигать элементы.
- Если выделено несколько элементов, перетаскивание любого выбранного элемента двигает весь набор.
- `Delete`: удалить выбранные элементы.
- Правая кнопка мыши по пустому месту: создать новый элемент в точке клика.
- Правая кнопка мыши по элементу: переименовать, дублировать, изменить цвета, сбросить цвета, удалить.
- Правая кнопка мыши по выбранному набору: сгруппировать, снять выделение или удалить выбранные элементы.
- `Картинка` / `Видео` / `URL`: добавить медиа-файл или встроенный веб-элемент как элемент доски.
- Для URL-элемента открывается отдельное окно ввода адреса; поддерживаются `http` и `https`.
- Для веб-элементов в режиме редактирования поверх сайта есть прозрачный слой для выбора и контекстного меню.
- Чтобы нажимать кнопки внутри сайта, включите `Взаимодействовать с сайтом` в контекстном меню URL-элемента или дважды щелкните по области сайта. Хедер останется доступен для перемещения и контекстного меню.
- `Esc`: выключить взаимодействие с сайтами и вернуть управление URL-элементами доске.
- `Группа`: добавить фоновую подложку. Карточки, центр которых находится внутри подложки, двигаются вместе с ней.
- `Настройки`: поменять тему интерфейса, язык, цвет фона, общий цвет соединений, цвета новых элементов, притягивание к сетке и hotkeys.
- Цвета в настройках применяются к новым элементам и к тем текущим элементам, которые еще не перекрашивались вручную.
- Цвета конкретного элемента меняются через контекстное меню.
- Хедер группы содержит drag-зону и поле названия; цвета и удаление доступны через контекстное меню.

## Запуск

```powershell
cd "E:\Coding Projects\desktop-board"
npm install
npm run dev
```

Если Electron пока не скачивается, можно проверить интерфейс в браузере без установки зависимостей:

```powershell
cd "E:\Coding Projects\desktop-board"
npm run preview
```

После этого открыть `http://127.0.0.1:5177`.

## Проверка кода без запуска Electron

```powershell
npm.cmd run check
```

## Сборка EXE

```powershell
cd "E:\Coding Projects\desktop-board"
npm.cmd run dist:win
```

Результат появляется в `dist`:

- `Desktop Board 0.1.1.exe` - portable exe без установки.
- `Desktop Board Setup 0.1.1.exe` - установщик Windows.
- `win-unpacked\Desktop Board.exe` - распакованная сборка для быстрой локальной проверки.

Сборка использует локальный Electron из `node_modules/electron/dist`, чтобы не скачивать Electron заново с GitHub. Для бинарников NSIS в npm-скрипте задано зеркало `electron-builder-binaries`.

Текущая сборка не подписана цифровым сертификатом и пока использует дефолтную иконку Electron. Windows SmartScreen может показывать предупреждение при запуске на другом устройстве.

## Следующие технические шаги

1. Добавить кастомную `.ico` и подключить ее к сборке.
2. Добавить tray icon, чтобы приложение было удобно вернуть после скрытия.
3. Сделать автозапуск вместе с Windows.
4. Добавить несколько рабочих пространств: `Сегодня`, `Проекты`, `Идеи`.
5. Проверить отдельный режим настоящего desktop/wallpaper-слоя на Windows.
6. Решить, нужен ли sync или все должно оставаться локально.
